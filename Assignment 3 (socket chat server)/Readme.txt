Usage:
The server should be started with node.js:
>node server.js

The server is hosted on port 2406 and reachable from a browser via the following request: http://localhost:2406/.

- Enter user name under prompt.
- A user can broadcast a message to all logged-in users in char room
- A user can send a private message to another user
- A user can block another user so no private message could be sent to.
- A user can unblock the previously blocked user so private message could be sent to again. 

Known Issues:
Assume username is unique.

Test under:
Windows, Chrome and node.js
